
#Retail-Pricing-Analysis-using-SQL

## Project Overview

This project analyzes Zepto's product inventory dataset using SQL to derive business insights related to pricing, inventory management, discounts, and product performance.

The objective is to perform data cleaning, exploratory data analysis (EDA), and business analysis using SQL queries to answer real-world business questions and support data-driven decision-making.

---

## Dataset Information

The dataset contains information about products available on the Zepto platform, including:

* Product Name
* Category
* Selling Price
* Market Price
* Discount Percentage
* Product Weight
* Available Quantity
* Stock Status
